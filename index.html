Quiz Builder

Welcome to the GIFT Question Creator

This tool allows you to quickly write quiz questions and export them in GIFT format, which is supported by Moodle for bulk question imports.

Simply choose the type, enter your questions into the form,(multiple choice, true/false, short answer, etc.), and add your answers and optionally feedback. When you're ready, you can export everything as a single GIFT file to upload into Moodle.
Make sure to use the radio buttons/checkbox to indicated which answer/s are correct.

Used for creating Moodle quizzes quickly without needing to write out the GIFT syntax yourself.

Tip: If you're not sure how GIFT works, check out this guide.
